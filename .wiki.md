# Project Summary
The RecruitReady project is an innovative fitness preparation platform designed to assist users in tracking their fitness progress and preparing for military assessments. It features cloud synchronization, subscription plans, a BMI calculator, and personalized onboarding. Users can monitor health metrics and receive tailored fitness advice based on their chosen military service branch. Recent updates include the integration of an Account page for managing personal information and subscription status, enhancing user experience.

# Project Module Description
- **Backend**: Handles database interactions, authentication, and API routes for user progress, preferences, and payment processing. The `payments.py` module supports direct Stripe payment links for subscriptions.
- **Frontend**: Provides a user-friendly interface for fitness tracking, viewing standards, managing subscriptions, onboarding, and now includes an Account page for profile management and subscription oversight.

# File Description Inventory
- **Backend**: Contains server-side logic, including API routes and database models. The `payments.py` file supports direct Stripe payment links for subscriptions.
- **Frontend**: Contains React components and pages, including onboarding, pricing, and a new Account page for user profile management and subscription details.

# Technology Stack
- **Backend**: Python, FastAPI, SQLAlchemy, Alembic, Stripe API
- **Frontend**: React, TypeScript, Tailwind CSS, Vite, pnpm

# Usage
To set up the project, follow these steps:
1. Install dependencies:
   - For the backend: Navigate to `app/backend` and run `pip install -r requirements.txt`.
   - For the frontend: Navigate to `app/frontend` and run `pnpm install`.
2. Build the frontend:
   - Navigate to `app/frontend` and run `pnpm run build`.
3. Start the backend server:
   - Run the backend application using the command `python main.py`.
